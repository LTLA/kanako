ArtifactDbSummarizedExperimentDatasetBase

ArtifactDbSummarizedExperimentDatasetBase

Dataset stored as a SummarizedExperiment in the ArtifactDB format. This is intended as a virtual base class; applications should define subclasses that are tied to a specific ArtifactDbProjectNavigator class. Subclasses should define the static format() and unserialize() methods.
